# folder-actions
An Apple Script that can organize specific folders

This script can help to mark all the new items in `~/Downloads/` folder `blue` tag, all the items exist more than 7 days `red` tag.

## Steps

1. Right click the target folder in Finder.
2. Select `Services->Folder Actions Setup`.
3. Check `Enable Folder Actions`.
4. Add target folder, like `Downloads` on left side.
5. New an `Apple Script` on right side.

Usually, the script will be located at this path: `~/Library/Scripts/Folder Action Scripts/auto_marker.scpt`.
